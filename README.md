#  Buget Lunar APP


A simple app helping to manage your monthly money.


## Content


**CRUD** operations:
    - **A**dd your money in the table
    - **R**ead transactions from DB
    - **U**pdate transactions in DB
    - **D**elete dates


## Setup

```
git clone https://github.com/domne21/buget-lunar-app.git
cd buget-lunar-app
npm install
```


## Running app

```
npm run devstart
```
